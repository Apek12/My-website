Plan Your Website Decide how many pages you need. Example: index.html → Home page about.html → About page contact.html → Contact page services.html → Services page --- 2. Create Each Webpage Write simple HTML for each page. Example: index.html Home Page

--- about.html About Us

About Us

Home | About | Services | Contact

(Do the same for services.html and contact.html.) --- 3. Link Pages Together Notice the
section — that’s where links to all pages are. You can copy-paste the same navigation menu on every page to make it easy to move around. --- 4. Folder Structure Keep your files in one folder like this: /my-website index.html about.html services.html contact.html --- 5. Open in Browser Just double-click index.html → It opens in your browser.
